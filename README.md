# Expenses Manager by AP

An intuitive and efficient Android application built to help users take control of their financial life. This app allows for seamless tracking of income and expenses with a focus on speed and user experience.

## ✨ Features
* **Daily Expense Tracking:** Quickly log expenses as they happen.
* **Category Management:** Organize spending into categories (Food, Transport, Rent, etc.).
* **Budget Summaries:** View daily, weekly, or monthly financial overviews.
* **Clean UI:** Material Design interface for a smooth user experience.
