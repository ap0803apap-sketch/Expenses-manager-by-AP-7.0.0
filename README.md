💰 Expenses Manager by AP

Expenses Manager by AP is an intuitive Android application designed to help users take control of their personal finances. The app focuses on simplicity, speed, and a clean user experience, making it easy to track daily spending and monitor financial habits.

✨ Features

💠Daily Expense Tracking: Quickly log expenses as they happen.

💠Category Management: Organize spending into categories such as Food, Transport, Rent, etc.

💠Budget Summaries: View daily, weekly, or monthly financial overviews.

💠Clean UI: Material Design interface for a smooth and modern experience.
